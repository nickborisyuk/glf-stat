Создай проект для учёта статистики ударов в гольфе.
Стек: Node.js (Express) + React (Vite или Create React App, на твой выбор).

Функциональность backend:
1. POST /rounds — создать новый раунд (поля: id, date, course).
2. GET /rounds/:id — получить данные по раунду.
3. POST /rounds/:id/holes/:holeId/shots — добавить удар (поля: shotNumber, club, distance, result).
4. GET /rounds/:id/stats — общая статистика по раунду (всего ударов, % удачных/неудачных).
5. GET /rounds/:id/stats/clubs — статистика по клюшкам (среднее расстояние, количество, % удачных/неудачных).

Функциональность frontend:
1. Экран выбора/создания раунда.
2. Экран списка лунок (1–18)
3. Экран отдельной лунки, где можно добавлять удары.
4. Форма добавления удара (селект клюшки, ввод расстояния, выбор результата).
5. Экран статистики:
   - Общая статистика по раунду.
   - Таблица по клюшкам: клюшка | удары | средняя дистанция | % успешных.
   - Графики (bar chart для дистанций по клюшкам, pie chart для % удачных/неудачных).

Данные можно хранить в памяти (массивы в JS) или в SQLite (на твой выбор).

Используй:
- Express для API.
- React + Tailwind для фронта.
- Recharts для графиков.
- Минимальная структура проекта, чтобы сразу можно было запустить и развивать.
